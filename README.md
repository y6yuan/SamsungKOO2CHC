# SamsungKOO2CHC
Samsung KOO firmware China Localization

## Features
* App lock
* PhoneCall Firewall
* Networkspeed Indicator
* Chinese Lunar in Calendar
* Chinese Lunar in Lockscreen 
* Yellow Page
* PhoneCall location
* Bixby Home CN
* PushService CN
* Bixby Touch
* TencentSecurityWifi

## Debloat List
* Korea Apps 
* Security Wifi
* SmartCallProvider
* HiyaService
* Samsung Pass
* AutoFramework
* ...
