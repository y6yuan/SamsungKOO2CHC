# SamsungKOO2CHC
Samsung KOO firmware China Localization
Test on Samsung S20 Ultra (sm-g988n)

## Features
* App lock
* PhoneCall Firewall
* Networkspeed Indicator
* Chinese Lunar in Calendar
* Chinese Lunar in Lockscreen 
* Yellow Page
* PhoneCall location
* Bixby Home CN
* PushService CN
* Bixby Touch
* TencentSecurityWifi

## Debloat List
* Korea Apps 
* Security Wifi
* SmartCallProvider
* HiyaService
* Samsung Pass
* AuthFramework
* ...
